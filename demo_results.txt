🚀 Demo 3: Getting Started
----------------------------
Steps to integrate AI into your hackathon project:

1. 📦 Install packages:
   pip install replicate python-dotenv requests

2. 🔑 Set up API access:
   - Visit https://replicate.com
   - Create account and get API token
   - Add token to .env file

3. 🧠 Use AI models:
   import replicate
   output = replicate.run('model-name', input={'prompt': 'your text'})

4. 🔧 Available models:
   - meta/llama-2-70b-chat (text generation)
   - salesforce/blip (image analysis)
   - stability-ai/stable-diffusion (image generation)
   - And many more at replicate.com/explore
